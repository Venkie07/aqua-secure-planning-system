## Project Overview

Rapid urbanization, climate change, and increasing pressure on water resources have made **water-related risks** like floods, groundwater depletion, and rainfall variability critical considerations in construction planning.  

The **Aqua Secure Planning System** is designed to help organizations, such as TATA, make **data-driven, sustainable, and future-resilient decisions** regarding site selection.

Key capabilities include:
- Evaluation of current water risk profiles for specific locations
- Predictive modeling of future water risks (2030, 2050)
- Disaster risk assessment for extreme events like floods
- AI-driven explainability and actionable recommendations
- Spatial suggestions for safer alternative sites

---

## Features

- **Project Input:** Specify location, type of construction (e.g., manufacturing, office, warehouse), and land characteristics.
- **Risk Assessment:** Integrates multi-source environmental and geospatial datasets:
  - Rainfall patterns
  - Flood susceptibility
  - Groundwater availability
  - Elevation
  - Land use
- **Composite Risk Score:** Weighted analysis combining multiple environmental factors.
- **Future Forecasting:** Predict water risk trends for 2030 and 2050.
- **Disaster Risk Module:** Identifies vulnerabilities to extreme rainfall, flooding, and groundwater stress.
- **Decision Intelligence Layer:** 
  - Human-readable explanations for risk factors
  - Suggests nearby safer alternative sites
- **Interactive Dashboard:** Visualizes:
  - Risk scores
  - Heat maps
  - Future projections
  - Disaster warnings
  - Recommendations

## Installation

1. Clone the repository:

```bash
git clone git@github.com:Venkie07/aqua-secure-planning-system.git
cd aqua-secure-planning-system
````

2. Create and activate a virtual environment:

```bash
python3 -m venv venv
source venv/bin/activate  # Linux/macOS
venv\Scripts\activate     # Windows
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
python app.py
```

5. Open the web dashboard in your browser:

```
http://localhost:5000
```
